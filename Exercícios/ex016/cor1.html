Exemplo de cores

No CSS, as cores são definidas por propriedades como color, para o texto, e background-color, para o fundo dos elementos. É possível representá-las de várias maneiras. Palavras-chave, como red ou blue, são simples e rápidas, mas limitadas. O formato hexadecimal, como #FF0000, permite esphecificar combinações de vermelho, verde e azul (RGB) com maior precisão. Existe também a forma reduzida, como #F00, equivalente a #FF0000. O formato RGB, como em rgb(255, 0, 0), usa valores numéricos para definir as cores e pode incluir transparência com o RGBA, por exemplo, rgba(255, 0, 0, 0.5), onde o último número ajusta a opacidade. Já o modelo HSL (Hue, Saturation, Lightness), como hsl(0, 100%, 50%), é mais intuitivo, permitindo ajustes de matiz, saturação e luminosidade, com a variante HSLA para incluir transparência.
