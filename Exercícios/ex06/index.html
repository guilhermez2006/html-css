Imagens

Imagens sem direitos autorais podem ser encontradas nos seguintes sites:

  • Pexels
  • Unsplash

Escolher o formato de imagem adequado depende do uso:

  • PNG: Ideal para imagens com fundos transparentes e para quando é necessário uma qualidade melhor (geralmente com um tamanho de arquivo maior).
  • JPG: Recomendado quando a compactação é importante, pois oferece uma boa relação entre qualidade e tamanho de arquivo.

Para ajustar e editar imagens, você pode utilizar o GIMP, uma ferramenta gratuita e poderosa para edição de imagens.
