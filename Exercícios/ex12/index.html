Cuidado com compatibilidade em diversos navegadores

A tag video permite incorporar vídeos diretamente no site. Ela suporta atributos como controls, autoplay, loop e muted.
Para formatos diferentes, usa-se a tag source dentro do video, garantindo a compatibilidade com navegadores.

Típos de vídeos

Os navegadores modernos suportam vários formatos de vídeo. Para garantir que seu vídeo funcione em todos os navegadores, é importante entender os principais tipos:
MP4: É o formato mais usado e amplamente compatível. Funciona na maioria dos navegadores e dispositivos,
sendo ideal para vídeos com boa qualidade e compressão.
WebM: É um formato moderno, eficiente e otimizado para a web.
É suportado por navegadores como Google Chrome e Firefox. É ideal para vídeos leves com alta qualidade.
Ogg: É um formato de código aberto, gratuito, mas com suporte limitado. Geralmente usado em navegadores que priorizam padrões livres.
