charset "UTF-8"/Global (*)/body:

@charset "UTF-8";: Define a codificação de caracteres da página como UTF-8, garantindo que caracteres especiais e acentos sejam exibidos corretamente.
Global (*): Define a fonte padrão para toda a página como "Arial, Helvetica, sans-serif", proporcionando uma aparência limpa e uniforme.
body: < Aplica um background com um gradiente linear que vai da cor laranja (#F2B749) ao tom alaranjado-avermelhado (#F28749), criando um fundo visualmente atraente

Main:

Main: Configura a área principal de conteúdo com: Fundo branco para destacar o texto e os elementos internos. Bordas arredondadas (border-radius: 10px) para um design mais suave. Sombra (box-shadow) para dar um efeito de elevação. Largura fixa de 600px e centralização (margin: auto). Espaçamento interno (padding: 7px) para manter o conteúdo bem posicionado.

H1/H2/P:

h1: Estiliza o título principal com: Cor laranja-avermelhada (#D9583B). Texto centralizado e sombra (text-shadow), conferindo destaque e profundidade.
h2: Define a cor dos subtítulos como um tom de azul escuro (#024959), contrastando bem com o fundo branco.
P: Ajusta o alinhamento do texto para "justify", garantindo uma distribuição uniforme das palavras, o que facilita a leitura.
