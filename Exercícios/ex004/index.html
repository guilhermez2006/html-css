Negrito / Destaque

Utilizar a tag <b> para negrito (semântica não indicada), ou a tag <strong> (semântica, indicada para ênfase forte, como quando um texto é realmente importante).

Itálico / Ênfase

Utilizar a tag <i> para itálico (semântica não indicada), ou a tag <em> (semântica, indicada para ênfase mais fraca, como para palavras que precisam de destaque).

Texto marcado:

Utilizar a tag <mark> para marcar partes do texto, normalmente para destacar um trecho que o usuário precisa perceber (como uma pesquisa).

Para mudar a cor da marcação, pode-se usar CSS, aplicando a propriedade background-color à tag <mark>.

Exemplo de alteração de cor de fundo:

        
            mark {
                background-color: yellow; /* ou qualquer cor desejada */
            }
