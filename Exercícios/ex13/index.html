Capítulo 1.1

CSS INLINE:
É a aplicação de estilos diretamente nos elementos HTML, por meio do atributo style. É útil para alterações rápidas e pontuais, mas não é recomendado para projetos grandes, pois dificulta a manutenção e reaproveitamento de estilos.!

Cápitulo 1.2

VANTAGENS:
Fácil e rápido para aplicar estilos pontuais. Útil para testes rápidos ou ajustes específicos. Não depende de arquivos externos, funcionando de forma imediata.
DESVANTAGENS:
Dificulta a manutenção em projetos maiores. Estilos não são reutilizáveis, gerando repetição de código. Pode deixar o HTML desorganizado e menos legível. Tem menor prioridade na hierarquia do CSS comparado a classes e IDs em arquivos externos.
