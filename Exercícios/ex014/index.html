Capítulo 1.1

CSS LOCAL (Interno):
É uma forma de aplicar estilos diretamente no arquivo HTML, dentro de uma tag style localizada no head. É útil para projetos pequenos ou páginas únicas, pois permite centralizar os estilos no mesmo documento.

Cápitulo 1.2

VANTAGENS:
Permite centralizar os estilos no mesmo arquivo HTML. Facilita a personalização rápida de uma página única. Não requer arquivos externos, funcionando de forma independente.
DESVANTAGENS:
Dificulta o reaproveitamento de estilos entre múltiplas páginas. Pode deixar o arquivo HTML mais longo e desorganizado em projetos maiores. Menos eficiente para manutenção em sites com várias páginas. Aumenta o tempo de carregamento se usado em excesso, já que estilos são carregados junto com o HTML.
