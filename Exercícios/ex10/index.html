Configuração Global

A configuração global no CSS, utilizando o seletor universal (*), permite aplicar estilos a todos os elementos de uma página de forma abrangente. Essa abordagem é muito útil para padronizar características iniciais de todos os elementos, como:

  • Margens: Remove as margens padrão de elementos.
  • Paddings: Remove o preenchimento padrão de elementos.
  • Bordas: Remove bordas padrões que podem ser aplicadas em alguns elementos.
  • Box-sizing: Define o comportamento da caixa de conteúdo.

Um exemplo de configuração global seria: 

        
            * {
                margin: 0;
                padding: 0;
                box-sizing: border-box;
            }

Essa configuração remove os espaços padrão dos elementos e garante que o modelo de caixa seja consistente em todos os navegadores, evitando problemas como o "deslocamento" de elementos. No entanto, é importante utilizar o seletor universal com cuidado, pois:

  • Desempenho: Pode impactar o desempenho em páginas complexas, pois aplica regras a todos os elementos da página.
  • Especificidade: O uso do seletor universal pode sobrescrever estilos mais específicos definidos para determinados elementos.

Id e Class

As tags id e class são formas fundamentais de selecionar e aplicar estilos específicos a elementos HTML:

1. id

O atributo id é único em uma página, ou seja, cada id deve ser atribuído a apenas um elemento HTML. Ele é utilizado para identificar elementos específicos e, geralmente, é usado para:

  • Aplicar estilos exclusivos: Como cada id é único, ele é muito eficaz para aplicar estilos específicos a um único elemento.
  • Facilitar a manipulação via JavaScript: Como o id é único, é fácil selecionar e manipular um único elemento com JavaScript.

Exemplo de uso do id: 

        
            #menu {
                background-color: #333;
                color: white;
            }

Neste exemplo, o id "menu" é usado para estilizar um elemento específico.

2. class

O atributo class pode ser atribuído a múltiplos elementos em uma página. Ele é ideal para quando você deseja aplicar o mesmo estilo a vários elementos. Ao contrário do id, que é único, o class pode ser reutilizado em diversos elementos.

  • Reutilização de estilos: Ideal para aplicar os mesmos estilos a múltiplos elementos.
  • Facilidade em combinar estilos: Você pode atribuir múltiplas classes a um único elemento, permitindo maior flexibilidade.

Exemplo de uso do class: 

        
            .botao {
                background-color: #008CBA;
                color: white;
                padding: 10px 20px;
                border: none;
                cursor: pointer;
            }

Neste exemplo, a classe "botao" pode ser atribuída a vários elementos, como botões, para dar a eles o mesmo estilo.
