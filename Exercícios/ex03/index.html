Abreviações

A tag <abbr> é usada para indicar abreviações ou siglas. Você pode usar o atributo title para fornecer o significado completo da palavra ou sigla. Quando o usuário passar o cursor sobre a abreviação, o significado será exibido.

Exemplo de abreviações comuns:

  • HTML: Linguagem de Marcação de Hipertexto, usada para estruturar páginas web.
  • CSS: Folhas de Estilo em Cascata, usadas para estilizar páginas web.
  • JSON: Notação de Objetos JavaScript, um formato de troca de dados leve e legível.

Além disso, a tag <abbr> também pode ser útil para tornar o conteúdo mais acessível, fornecendo descrições rápidas para siglas e abreviações que podem não ser compreendidas por todos os usuários.
